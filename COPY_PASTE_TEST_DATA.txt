================================================================================
HOW TO USE:
================================================================================

1. Open n8n: http://localhost:5678
2. Create new workflow
3. Add "Set" node
4. Copy one test case (JSON object) into Set node
5. Add FAIM Forecast node
6. Configure according to test instructions
7. Click "Execute Workflow"
8. Check "Results" tab for output

For error tests, you should see the error message in the execution panel.
